### INSTALLATION 
* sudo apt-get install proxychains
* git clone https://github.com/0xbitx/Dedsec-Proxychain.git
* cd Dedsec-Proxychain
* chmod +x dedsec-proxychain.py
* sudo python3 dedsec-proxychain.py
